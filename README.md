# CASH+ME+IF+U+CAN
Xinyu Zhou xzhou020@gold.ac.uk
Xinchun Fu xfu002@gold.ac.uk

## Short Description
This project is inspired by the classic game Gold
Miner and Mario. Players tilt their phones left, right,
forward, and backward to control the character’s
movement, while a mouth on the screen extends a
tongue to grab coins. The coins appear randomly on
the screen, and the length of the tongue changes in
real time according to how much the player moves
the phone, affecting the distance and rhythm of the
grab. The work also includes a multiplayer scoring
system: participants can see the number of coins they
have collected on their own screen, as well as the total
collected by all players.

## How to Run / Install
Use your phone to open the link: https://cash-me-if-u-can.onrender.com
